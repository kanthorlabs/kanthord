# Implementation cycle — 019.5-task-audit-timeline

Pulled from EPIC: `.agent/plan/epics/019.5-task-audit-timeline.md`.

Verification gate (binding, from the EPIC's `## Verification Gate` section):
> - `npm run typecheck` exits 0; `npm test` green — new session-event, timeline, per-call, attribution, and query suites plus all pre-existing suites, no regression. Zero-network guard stays green (session events + model calls run on doubles).
> - **Session events:** a spawned session (double) emitting tool-call/message/usage/error events surfaces them through `PiSessionHandle` as one ordered read-only stream; the daemon records them; no inbound extension point is exposed.
> - **Timeline reconstructs:** given a scripted task run (dispatch → spawn → tool call → broker op → gate fail → escalation → respawn), `queryTaskTimeline(taskId)` returns one ordered timeline whose events all share the task's `correlation_id`, spanning attempt, session (child), broker op, and gate/interaction records — reconstructed from **durable state only** (the LP7 "audit reconstruction" drill, now automated).
> - **Root-cause step is identifiable:** for each of the machine signals (`rate_limited / quota_exhausted / auth_failed / tool_blocked / budget_breach / broker_failed / gate_failed`), a scripted failure produces a timeline whose failing event carries that `observed_failure_signal` and, for a model-call failure, the `account_id` + `model` that served it.
> - **Honest attribution:** a prompt/model-behavior failure is **not** auto-classified — it records a factual signal (e.g. `gate_failed`) with `suspected_root_cause` left for human confirmation; a test asserts no machine code writes `suspected_root_cause`.
> - **Multi-account:** two attempts of one task on two different accounts produce timeline events each attributed to the correct `account_id`.
> - **CLI end-to-end (hermetic):** the minimal CLI reads a task timeline and filters to the failing step, without network.
> - **Gate closes when** the above are green and the reconstruction drill passes on durable records alone.
